Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The br element creates a line of empty space
  • The img element contains information about images that are displayed
  • the ul, ol, and li represent unordered lists, ordered lists and list items

CSS

  • a link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location (URL) of the external resource
  • The href attribute is using a relative path. This means the location is derived from the current location

git-section

  • When you create a repository on GitHub.com, it exists as a remote repository. You can clone your repository to create a local copy on your computer and sync between the two locations
  • a default branch is called main, but master may also be seen in older repos
  • We shouldn't code directly under the main branch
  • When we create a new branch to add something new to our project, we call it a feature branch
  • A pull request, or PR, is used to merge a comparison branch
